No CSS nós possuímos os seletores por ID

  • Seletor por tipoBusca elementos por uma tag HTML específica
  • Seletor por IDBusca elementos através do atributo "id"
  • Seletor por classeBusca elementos através do atributo "class"
  • Seletor universalSeleciona todos os elementos HTML
  • Seletor de atributoSeleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos também buscar atributos com um valor específico
